# weatherdashboard

## Description

Weather dashboard application created using HTML, CSS, Bootsrap, JQuery, and JavaScript.


## Features

* Current weather conditions
* Five day forecast
* Search saved in local storage
* Displays UV index, wind speed, humidity, and temperature
